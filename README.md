# Stress Management System

## Overview

The Stress Management System is a web-based application designed to help users assess their stress levels and receive personalized recommendations for improving their mental well-being. The platform combines stress assessment surveys, personalized suggestions, relaxation activities, and stress tracking to promote healthier lifestyle habits.

## Features

### User Authentication

* User Registration and Login
* Secure user account management
* Personalized user experience

### Stress Assessment Survey

* Interactive questionnaire to evaluate stress levels
* Multiple lifestyle and behavioral indicators
* Automated stress score calculation

### Personalized Recommendations

Based on the user's stress score, the system provides:

* Stress level classification (Low, Moderate, High)
* Customized wellness recommendations
* Daily self-care guidance
* Stress reduction techniques

### Relaxation Activities

The platform includes interactive relaxation games to help users reduce stress:

* Sliding Puzzle Game
* Memory Match Game

### Stress History Tracking

* Store previous assessment results
* Monitor stress trends over time
* Track personal progress

### Calming Music Support

* Relaxing audio recommendations
* Stress-relief music integration
* Enhanced relaxation experience

## Workflow

1. User registers or logs into the platform.
2. User completes the Stress Management Survey.
3. The system calculates the stress score.
4. Personalized recommendations are generated.
5. Users can access relaxation games and calming music.
6. Stress history is stored for future reference and progress tracking.
